# mediawiki

mediawiki searcher

**Version:** 1.0.0
**Author:** himeno
**License:** GPL-3.0

## List of command

-   /wiki <content> <num?> - Search MediaWiki, specify target Wiki with num
-   /wikil - View MediaWiki list
-   /wikio add <name> <url> - Add MediaWiki^
-   /wikio del <name> - Delete MediaWiki^

## Lang Support

-   ja_JP
-   en_US
-   zh_TW
-   zh_CN
